# 播客 APK 安装指南

## 📱 安装方法

### 方法1：直接安装（可能需要权限）
1. 在手机文件管理器中找到 `/sdcard/OpenClaw/PodcastKotlinApp_basic.apk`
2. 点击安装
3. 如果提示"未知来源"，请在设置中允许安装未知应用

### 方法2：使用 Android Studio（推荐）
1. 将整个项目文件夹传输到电脑
2. 在 Android Studio 中打开项目
3. 点击 "Build" -> "Build APK"
4. 将生成的 APK 传输回手机安装

### 方法3：网页版本（立即可用）
1. 打开手机浏览器
2. 输入：`file:///sdcard/OpenClaw/podcast_player.html`
3. 直接使用，无需安装

## 🔧 如果 APK 无法安装

### 原因分析
1. **未签名**：基本 APK 框架没有签名
2. **兼容性**：可能与手机系统不兼容
3. **权限问题**：需要开启"未知来源"安装权限

### 解决方案
1. **使用网页版本**：功能完整，立即可用
2. **完整构建**：使用 Android Studio 重新构建
3. **在线构建**：使用在线 Android 构建服务
